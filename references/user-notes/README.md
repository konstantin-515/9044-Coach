# User Notes

把你自己的复习笔记放在这里，`9044-coach` 之后出题时就可以参考它们。

## 目录

- `md/`：你自己写的 Markdown 总结
- `pdf/`：课程 PDF、整理过的 PDF 笔记

## 建议

- 能写成 Markdown 的尽量放到 `md/`
- PDF 放到 `pdf/`
- 如果你希望出题时更贴近你的复习重点，可以直接告诉 agent “参考我的笔记”

## 例子

- `md/grep-notes.md`
- `md/regex-mistakes.md`
- `pdf/week05-summary.pdf`
- `pdf/midterm-review.pdf`
